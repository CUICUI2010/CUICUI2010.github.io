CF Round #804 题解

题目质量好，思维题多。

A

思路

注意到答案如果为奇必定无解。

证明：

  • 引理：$(a \oplus b) \bmod 2 = (a + b) \bmod 2$。证明略。
  • $a + b + b + c + c + a
    = 2 \times a + 2 \times b + 2 \times c
    = 2 \times(a+b+c)$，显然为偶。

如果为偶，设此数为 $x$。则输出两个 $\frac{x}{2}$ 和一个 $0$ 即可。

Code

B

思路

构造。方案（且疑似唯一的方案）如下：

  • 当 $x \bmod 4 = 1$ 或 $x \bmod 4 = 3$ 时，

    • 当 $y \bmod 4 = 1$ 或 $y \bmod 4 = 3$，输出 1，否则输出 0。

  • 否则，

    • 当 $y \bmod 4 = 1$ 或 $y \bmod 4 = 3$，输出 0，否则输出 1。

另一种方案与其相反，算作同一种。

此构造方法画图暴做即可得到。

Code

C

数学。废话

不难发现，$0$ 和 $1$ 的位置必须和原来的位置相同。

其实，由此可得到正解。

  • 设 $0$ 在 $a$ 的位置为 $x$，$1$ 在 $a$ 的位置为 $y$。
  • 则如果 $0$ 的位置不一样，我们可以用 $[x,x]$ 的区间把这个方案炸掉。
  • 如果 $1$ 的位置不一样（设它为 $z$），我们可以用 $[x,y]$ 或 $[x,z]$ （如果 $x > z$ 就反过来）其中之一炸掉。

我们可以由此得到我们的正解。设 $i$ 为当前的数，$p_i$ 为当前数所在的位置。

  • 设 $l, r$ 为当前小于 $i$ 的所有数所在的最小区间。
  • 如果 $p_i$ 在 $[l,r]$ 外，则 $i$ 不可移动。反例和 $1$ 的反例一样。
  • 否则，$i$ 可以在区间里随意动。因此将答案乘上区间内剩余的位置数即可。证明略。

注意，一定要 $\bmod 10^9 + 7$ ！
